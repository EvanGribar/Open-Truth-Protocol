# Quick Start Guide

**Open Truth Protocol** is an open-source initiative to achieve **100% accuracy in detecting AI-generated text**.

This is a research project. We provide specifications, methodologies, and benchmarks. Implementation is ongoing.

## ❓ FAQ

**Q: Is there code to run yet?**  
A: Not yet. We're in Phase 1 (signal research). Code comes in Q3 2026.

**Q: Can I help?**  
A: Yes! See [CONTRIBUTING.md](CONTRIBUTING.md).

**Q: What's with all the old agent code?**  
A: Legacy from the previous multi-modal architecture. Archived for reference. See [AGENTS.md](AGENTS.md).

**Q: When is v1.0 release?**  
A: End of Q4 2026 (Phase 3 completion + validation gates).

**Q: Is this only for English?**  
A: Phase 1 is English-only. Multilingual support planned for Phase 4 (2027).
